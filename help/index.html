A) Intro - top

These buttons are fully customizable, just playing with their classes. As its name suggests, there are more than 360 possibilities (without considering the color variation). Their effects don't use image and js, they are 'pure CSS'.

B) Requirements - top

To work properly, these buttons need some requirements :

  • Call main.css and color.css in your head tag
  • Of course, if you want an accurate look, don't override my CSS.
  • On the other hand, to not override your own CSS, please add the class .holy * to my buttons
  • These buttons can be a button tag, an a tag, and a span
  • Just follow the rules below ;-)

* "Holy" is the name of my adorable dog

C) Variation - top

The following variations can be used in each style which I'll explain in the (D) part.

1) Colors

There are 10 colors defined : red, orange, yellow, blue, green, turquoise, pink, purple, grey and black. To use them, just add the class corresponding to their name :

<button class="holy simple red">Red</button>
<button class="holy simple orange">Orange</button>
<button class="holy simple yellow">Yellow</button>
<button class="holy simple blue">Blue</button>
<button class="holy simple green">Green</button>
<button class="holy simple turquoise">Turquoise</button>
<button class="holy simple pink">Pink</button>
<button class="holy simple purple">Purple</button>
<button class="holy simple grey">grey</button>
<button class="holy simple black">Black</button>

Notice the class .simple. It's just the class used for the default style. Don't worry, I'll mention below later.

2) Size

There are 4 sizes available : small, medium, normal (default size, it doesn't need a class) and big. To use them, just add the class corresponding to their name :

<button class="holy simple blue small">small</button>
<button class="holy simple blue medium">medium</button>
<button class="holy simple blue">normal</button>
<button class="holy simple blue big">big</button>

3) Corner radius

There are 4 corner radius styles : none (default, it doesn't need a class), light-radius, normal-radius, round-radius. To use them, just add the class corresponding to their name :

<button class="holy simple blue">no radius</button>
<button class="holy simple blue light-radius">light radius</button>
<button class="holy simple blue noraml-radius">normal radius</button>
<button class="holy simple blue round-radius">round radius</button>

4) Icons

These buttons can be accompanied by an icon, or have only an icon as content. To use them, please refer to Foundation Icon Fonts 3 to know the class corresponding to the class you need. (Of course, you can use your own font icon by changing the font files) For example, if I need a button with a star as icon and another button with a minus as icon, it will be :

<button class="holy simple blue fi-star">your text</button>
<button class="holy simple blue fi-minus">your text</button>

If i need buttons with icons but not text, don't put text obviously, and just add the class .only-icon. Example : 

<button class="holy simple blue fi-star only-icon"></button>
<button class="holy simple blue fi-minus only-icon"></button>

Be aware that some styles need icons to work properly. Don't worry, I'll mention below later.

D) Button styles - top

#1 style : Simple

The first, called "Simple", is the default style. No special things, just add the class .simple.

<button class="holy blue simple">your text</button>

Example, I need a big (.big) red (.red) round (.round-radius) button, using the "Simple" style (.simple) :

<button class="holy simple red big round-radius">your text</button>

#2 style : White Blow

The second style, called "White Blow" ( try to guess the reference ;-) ), honors the white and has 3 cool hover effects.
To use "White Blow", add the class .wb : 

<button class="holy blue wb">your text</button>

For the first hover effect, add one of following classes : .hover1-top, .hover1-right, .hover1-bottom, .hover1-left, .hover1-topleft, .hover1-topright, .hover1-bottomright, .hover1-bottomleft, .hover1-center. Please, note that hover effect needs a <span> as child.

<button class="holy blue wb hover1-top"><span>To the top</span></button>
<button class="holy blue wb hover1-right"><span>To the right</span></button>
<button class="holy blue wb hover1-bottom"><span>To the bottom</span></button>
<button class="holy blue wb hover1-left"><span>To the left</span></button>
<button class="holy blue wb hover1-topleft"><span>To the top left</span></button>
<button class="holy blue wb hover1-topright"><span>To the top right</span></button>
<button class="holy blue wb hover1-bottomleft"><span>To the bottom left</span></button>
<button class="holy blue wb hover1-bottomright"><span>To the bottom right</span></button>
<button class="holy blue wb hover1-center"><span>From the center</span></button>

For the second hover effect, add the class .hover2.

<button class="holy blue wb hover2">your text</button>

For the third hover effect, add the class .hover3. Please, note that hover effect works only with icon+text

<button class="holy blue wb hover3 fi-check">your text</button>

#3 style : 3 Dimensions

The third style, called temperately "3 Dimensions", contains three different rendering of 3D perspective.

To use the first style of "3 Dimensions", add the class .threeD : 

<button class="holy blue threeD">your text</button>

To use the second style of "3 Dimensions", add the classes .threeD and .style2 : 

<button class="holy blue threeD style2">your text</button>

To use the third style of "3 Dimensions", add the classes .threeD and .style3 : 

<button class="holy blue threeD style3">your text</button>

#4 style : Gym'Icon

The fourth style, called funnily "Gym'Icon". As its name suggests, this style makes living icons. (Obviously, the following styles don't work without an icon)

To rotate an icon when hover, add the classes .gymicon and .rotate360 (or .rotate180 or .rotate90)

<button class="holy blue gymicon rotate360">360°</button>
<button class="holy blue gymicon rotate180">180°</button>
<button class="holy blue gymicon rotate90">90°</button>

To move an icon when hover, add the classes .gymicon and .move-left (or .move-top or .move-bottom)

<button class="holy blue gymicon move-left">your text</button>
<button class="holy blue gymicon move-top">your text</button>
<button class="holy blue gymicon move-bottom">your text</button>

To burst an icon when hover, add the classes .gymicon and .burst (or .move-top or .move-bottom)

<button class="holy blue gymicon burst">your text</button>

To apply a loop animation, add the classes .gymicon and .spin (or .levitate or .shake or .fade)

<button class="holy blue gymicon spin">your text</button>
<button class="holy blue gymicon levitate">your text</button>
<button class="holy blue gymicon shake">your text</button>
<button class="holy blue gymicon fade">your text</button>

To apply a loop animation when hover, add the classes .gymicon, .hover and .spin (or .levitate or .shake or .fade)

<button class="holy blue gymicon spin hover">your text</button>
<button class="holy blue gymicon levitate hover">your text</button>
<button class="holy blue gymicon shake hover">your text</button>
<button class="holy blue gymicon fade hover">your text</button>

#5 style :Vertical

The last style (phew...), foolishly called "Vertical", is for... vertical buttons. There are 2 styles : plain et realistic.
To use this style, add the classes .simple, .vertical, and .style1 (or .style2). These styles need an icon and a <span> as child.

<button class="holy blue simple vertical style1 fi-widget"><span>your text</span></button>
<button class="holy blue simple vertical style2 fi-pencil"><span>your text</span></button>

Create your own buttons by mixing the classes ! Without count the colors, there are over 360 availables combinations.
